Impresora 3d compuesta por elementos reciclados

(en desarrollo)

Este proyecto consiste en el armado y puesta en funcionamiento de una impresora 3d utilizando un 80% de elementos reciclados y caceros. Para la misma se utilizan motores, rieles, cintas y otros componentes de diversas impresoras de oficina obsoletas y sin uso. La estructura de la impresora 3d esta realizada con varillas roscadas, tuercas, arandelas, madera mdf pintada con pintura sintética y otras partes creadas por otra impresora 3d. Para el sistema de control y lógica se utiliza una placa Arduino Mega y Ramps con drivers para los motores. Llevo un tiempo trabajando en este proyecto y aunque se presentaron diversas dificultades estoy cerca de terminar para hacer las primeras impresiones de prueba.
